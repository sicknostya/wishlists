# wishlists
Web service for creating and managing wishlists

## for frontend:

- install node.js 22.12.0
- cd frontend
- npm i
- npm start

app will starts at http://localhost:3000

## for backend:

- install python 3.13.2
- cd backend
- python manage.py runserver

app will starts at http://127.0.0.1:8000/
